# Python Boilerplate
Simple boilerplate for python project which are independent of frameworks and contains scripts and crons. 

## This boilerplate contain

1. Travis
2. Config
3. mysql Connection
4. fabfile
5. Dockerfile
6. Beanstalk setting
7. License
8. Logger
